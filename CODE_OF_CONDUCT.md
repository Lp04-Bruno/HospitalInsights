# Code of Conduct

## Unser Anspruch

Hospitalinsights soll ein respektvoller, sachlicher und konstruktiver Ort für
Zusammenarbeit sein. Unterschiedliche Erfahrungsstände, Hintergründe und
Perspektiven sind willkommen.

## Erwartetes Verhalten

- respektvoll und freundlich kommunizieren;
- Kritik konkret, sachlich und lösungsorientiert formulieren;
- andere nicht herabsetzen oder persönlich angreifen;
- unterschiedliche Meinungen ernst nehmen;
- vertrauliche Informationen und Sicherheitsdetails verantwortungsvoll
  behandeln.

## Nicht akzeptiertes Verhalten

- Beleidigungen, Belästigung, Drohungen oder Diskriminierung;
- sexualisierte, abwertende oder einschüchternde Sprache;
- Veröffentlichung privater Informationen ohne Zustimmung;
- absichtliches Stören von Diskussionen, Issues oder Pull Requests;
- Missbrauch von Sicherheitsmeldungen oder vertraulichen Informationen.

## Durchsetzung

Maintainer können Beiträge bearbeiten, ausblenden, schließen oder Personen von
der Teilnahme ausschließen, wenn Verhalten diesem Code of Conduct widerspricht.

Probleme können über GitHub an den Maintainer `@Lp04-Bruno` gemeldet werden.
Sicherheitsrelevante Inhalte gehören zusätzlich nach [SECURITY.md](SECURITY.md)
nicht in öffentliche Issues.
